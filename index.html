Projects

1. Data Link & Physical Layer Simulator

Developed a network simulator in Python to model the Physical and Data Link layers of the OSI model. Implemented bit-level transmission, framing, error control, flow control using Go-Back-N (GBN), and access control using CSMA/CD for simulating real-world data communication protocols.

2. Prediction of Horticulture Crops.

Developed a time series forecasting model using ARIMA and SARIMA to predict the demand of horticulture crops in India till 2035. The project analyzes historical data trends to generate accurate and insightful future projections for agricultural planning.

3. Prediction of protein Methylation sites.

This project uses ESM encoding and a CNN model to predict protein methylation sites. It aims to identify potential modifications in protein sequences, helping enhance our understanding of protein function and regulation.
